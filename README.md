# 🌐 Personal Portfolio (Django)

A modern and responsive **portfolio website** built using Django to showcase my skills, projects, and experience as a **Data Analyst**.

---

## 📌 Features

* 👨‍💻 Personal introduction (Hero section)
* 📊 Projects showcase
* 🧠 Skills section
* 📬 Contact functionality
* 🖼️ Media & static file handling
* ⚡ Clean and responsive UI

---

## 🛠️ Tech Stack

* **Backend:** Django
* **Frontend:** HTML, CSS, JavaScript, Bootstrap
* **Database:** SQLite
* **Version Control:** Git & GitHub

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/personal-portfolio.git
cd personal-portfolio
```

### 2️⃣ Create Virtual Environment

```bash
python -m venv venv
```

Activate:

```bash
venv\Scripts\activate   # Windows
source venv/bin/activate  # Mac/Linux
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run Migrations

```bash
python manage.py migrate
```

### 5️⃣ Start Server

```bash
python manage.py runserver
```

Visit 👉 http://127.0.0.1:8000/
